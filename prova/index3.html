resumo: HTML e CSS

Bom,para entendermos melhor esses asuuntos,vamos pensar que o html é uma casa,mas sem a pintura,decoração,apenas a estrutura,parede,teto,chão, no html podemos adicionar imagens,links,vários tipos de lista,tabelas,paragráfos,titulos e muito mais. O html é basicamente isso,uma página web só com a estrutura:

  • body ("corpo" da página)
  • head ("cabeça" da página)
  • p ( indica um paragrafo na página)
  • h (para colocar um titulo no texto da sua página)
Já o CSS é toda essa "estilização" da sua página na web,por exemplo,mudar cor de fundo,mudar cor de texto,definir tamanho da fonte. Nós estudamos 3 tipos de CSS o inline,interno e externo. o inline é aquele que fica dentro da propria tag, por exemplo: body = style "background-color: pink;". O css interno também vem dentro da página,mas vem separado das tags da estrutura, e o externo é o que você precisa fazer um novo arquivo só pra ele,ou seja,fora da página.Mas tem uma tag que "liga" esses dois arquivos,é: link rel = "stylesheet" href = "style.css".
